## Why

To build real, working agentic AI applications in Python — starting from zero — and understand how they map to the Java world you already know. The goal is working software, not theory.

## Success looks like

- Call the Claude API from Python without looking anything up
- Build an agent that uses tools (web search, file read, calculator) autonomously
- Understand the agent loop well enough to debug it when it goes wrong
- Wire a multi-agent system where agents hand off work to each other
- Map every core Python pattern to its Java Spring Boot equivalent
- Ship at least one complex agentic app to production (or a demo)

## Constraints

- Primary language: Python (Jupyter Notebooks for simple lessons, Python projects for complex apps)
- Java used as a conceptual bridge — not to be built, just to relate
- Lessons should be completable quickly (working memory is small)
- Step-by-step HTML instructions required for each lesson

## Out of scope

- Deep ML/training (we use hosted models, not train our own)
- LangChain or LlamaIndex (we learn the primitives first)
- Frontend/UI (unless needed to demonstrate an app)
- Java Spring Boot implementation of agents (Java is the reference frame, not the target)
