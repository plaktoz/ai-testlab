# Session Start — Goals and Background Established

User is a Java developer learning Python and Agentic AI from scratch.
Goal: build real working agentic apps, starting simple (Jupyter) and progressing to complex Python projects.
Java is a conceptual bridge, not a target — user has strong Java intuition that can be leveraged throughout teaching.

## Implications for future sessions

- Always provide a Java comparison for every new concept introduced
- Jupyter notebooks are the format for simple/moderate lessons
- Keep lessons short and skill-focused — one win per lesson
- User wants step-by-step HTML instructions; do not assume they will figure out the steps
- Complexity order: simple Jupyter → moderate Jupyter+scripts → complex Python project
- Start with Haiku model (fast, cheap) — only upgrade when the lesson requires more reasoning
