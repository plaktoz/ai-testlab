# Resources

## Knowledge

- **Anthropic Python SDK Docs** (Anthropic) — Official reference for `anthropic` Python package. The ground truth for API calls, tool use, streaming, and message formats. https://docs.anthropic.com/en/api/getting-started
- **Claude Model Overview** (Anthropic) — Which model to pick (Haiku vs Sonnet vs Opus), pricing, context windows. https://docs.anthropic.com/en/docs/about-claude/models/overview
- **Building with Claude — Tool Use Guide** (Anthropic) — The definitive guide to function calling / tool use. Covers input_schema, multi-tool, parallel tool use. https://docs.anthropic.com/en/docs/build-with-claude/tool-use/overview
- **Agentic AI Overview** (Anthropic) — Anthropic's own framing of agents, orchestrators, and the agent loop. https://docs.anthropic.com/en/docs/build-with-claude/agents/overview
- **Jupyter Notebook Beginner Guide** (Jupyter Project) — Environment setup and notebook basics. https://jupyter-notebook.readthedocs.io/en/stable/notebook.html
- **Anthropic Java SDK** (Anthropic on Maven Central) — Java SDK for the Claude API (group: `com.anthropic`, artifact: `anthropic-java`). Use as Java reference throughout lessons. https://github.com/anthropics/anthropic-sdk-java

## Wisdom (Communities)

- **Anthropic Discord** — Active channel #api-and-sdk; good for SDK-specific questions and model behaviour edge cases.
- **r/ClaudeAI** (Reddit) — Community of Claude users and builders; real-world use cases and gotchas.
- **Hugging Face Forums** — Broader AI practitioner community; good for multi-agent patterns and evaluation approaches.
